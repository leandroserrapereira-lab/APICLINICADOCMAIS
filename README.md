git config --global user.name "Seu Nome": Define seu nome de usuário.

Iniciando um Repositório:
git init: Inicializa um repositório local.
git clone <url>: Clona um repositório remoto.
Fluxo de Trabalho (Staging e Commit):
git status: Verifica o estado dos arquivos (modificados, staged).
git add <arquivo>: Adiciona um arquivo específico à área de preparo (staging).
git add .: Adiciona todos os arquivos modificados.
git commit -m "mensagem": Grava as alterações do staging no histórico local.
Ramificação (Branching):
git branch: Lista, cria ou deleta branches.
git checkout -b <nome-branch>: Cria e muda para um novo branch.
git merge <branch>: Mescla um branch ao atual.
Repositório Remoto (GitHub):
git push origin <branch>: Envia seus commits locais para o GitHub.
git pull: Traz e integra as mudanças do repositório remoto.
Desfazer Alterações:
git reset --hard: Desfaz alteraçõ
